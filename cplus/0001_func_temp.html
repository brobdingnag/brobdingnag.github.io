以下は足し算addを多重定義（オーバーロード）した例である。 

int add(int a,int b) {
    return a+b;
}
double add(double a,double b) {
    return a+b;
}
次に、addをひとまとめにした関数テンプレートを定義した。 

template <class T>
T Add(T a,T b){
    return a+b;
}
呼び出しもとに応じて、Tの部分にintやdoubleが代入されると考えればよい。 次は、テンプレートの例外処理を示す。 

template <class T>
T Add(T a,T b){
    return a+b;
}

template <>
int add(int a,int b){
    return a + b + 1;
}
上のコードでは、整数を引数にとった時だけ、2数の和に1を加えて返すことになる。
